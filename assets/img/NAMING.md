# Bilder für die Webseite

Lege die Fotos hier im Ordner `assets/img/` mit **genau diesen Dateinamen** ab.
Danach erscheinen sie automatisch in Galerie und Hero-Hintergrund – ohne weitere Änderungen.
Fehlt ein Foto, zeigt die Galerie einen dezenten Platzhalter „Foto folgt" (kein kaputtes Bild).

| Dateiname | Motiv (so hast du es geschickt) |
|-----------|--------------------------------|
| `lager-europaletten.jpg` | Große Lagerhalle voller gestapelter Europaletten (auch Hero-Hintergrund) |
| `lager-holzkisten.jpg` | Gestapelte Aufsatz-/Holzkisten in der Halle |
| `produktion-paletten.jpg` | Stapel neuer Paletten in der Werkstatt |
| `aufsetzrahmen.jpg` | Aufsetzrahmen / Gitteraufsatz auf Palette (mit Gabelstapler) |
| `sonderpalette.jpg` | Lange, kräftige Sonderpalette (Nahaufnahme) |
| `dj-paletten-detail.jpg` | Nahaufnahme Paletten mit „D.J. PALETTEN"-Kennzeichnung |

Optional:

| Dateiname | Motiv |
|-----------|-------|
| `logo.png` | Firmenlogo (transparenter Hintergrund). Sag Bescheid, dann setze ich es statt des nachgebauten SVG-Logos ein. |

## Hochladen über GitHub (einfachster Weg)

1. Im Repository in den Ordner **`assets/img/`** gehen.
2. **Add file → Upload files** anklicken.
3. Fotos hineinziehen, exakt wie oben benennen.
4. Auf den Branch `claude/create-website-hwikle` committen.

Bilder möglichst auf ca. 1600 px Breite verkleinern und als `.jpg` speichern,
damit die Seite schnell lädt.
